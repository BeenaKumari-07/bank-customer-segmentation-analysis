# Bank Transaction & Customer Segmentation Analysis

## 📊 Project Overview
This project provides a comprehensive analysis of a retail bank's customer base and transaction patterns. By leveraging data visualization, the project identifies key demographics, spending behaviors, and high-value customer segments.

---

## 🚀 Key Insights

### 1. Financial Health & Scale
*   **Total Customer Base:** 880,000 customers managing over **119.82bn** in total balances.
*   **Transaction Volume:** 1M transactions totaling **1.63bn** in value.

### 2. Customer Segmentation (Value-Based)
*   **The Pareto Principle:** While "Premium Customers" represent only ~11% of the population, they hold **76.5% of the total bank balance** (91.76bn).
*   **Volume vs. Value:** Low-value customers drive the highest transaction volume but contribute the least to total balance.

### 3. Demographic Analysis
*   **Gender Gap:** The customer base is heavily skewed (71.4% Male).
*   **Age Dynamics:** The 36-50 age group is the largest, but the **18-25 group** maintains the highest average balance.

---

## 📈 Strategic Recommendations
*   **Retention:** Implement loyalty programs for "Premium" segments to protect 76% of assets.
*   **Growth:** Target the 18-25 demographic for high-liquidity potential.
*   **Inclusivity:** Market to female customers to diversify the current 28% base.
